📬 GPREC Suggestion Box - Admin Panel

This is a lightweight, client-side admin panel for managing digital suggestions submitted by students or users. Built using HTML, CSS, and JavaScript, it allows admin users to view, filter, manage, and export suggestions.

✨ Features

- 🔐 Password-protected admin access
- 📊 Status dashboard (Pending, Reviewed, etc.)
- 🔎 Search & filter by status
- 📝 Add notes per suggestion
- 🔁 Update status (Pending, Reviewed, Implemented, etc.)
- 🧹 Delete individual or all suggestions
- 📄 Export suggestions as CSV
- 📦 Suggestions stored in browser's LocalStorage
- 📱 Responsive design for all devices

🚀 How to Use

1. Clone/download this project.
2. Open `index.html` in your browser.
3. Enter the admin password: admin@2025
4. Start managing suggestions!
